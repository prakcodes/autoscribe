# AI Notability - Smart Note Taking

An intelligent note-taking application with AI-powered content generation. Write prompts and let AI automatically generate structured notes, meeting summaries, todo lists, essays, and more.

## Framework

Vite 6.x + React 19.x

## Features

- **AI Content Generation**: Describe what you need and AI generates complete, formatted content
- **Smart Templates**: Quick templates for meetings, todos, essays, summaries, and more
- **Multiple Insert Modes**: Replace, append, or prepend AI-generated content
- **Real-time Preview**: See formatted markdown preview of your notes
- **Local Storage**: All notes saved automatically in your browser
- **Clean Interface**: Modern, intuitive design with sidebar navigation

## Theme

Clean Light theme with white/gray backgrounds, blue accent colors, and gradient purple-to-blue AI features. Soft shadows and rounded corners create a modern, professional aesthetic.
